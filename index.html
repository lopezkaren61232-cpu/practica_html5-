A diferencia de versiones antiguas, HTML5 se diseñó pensando en ser más flexible, potente y compatible con cualquier dispositivo. ¿Qué es HTML5?

1. ¿Qué es HTML5? Definición HTML5 es la quinta versión oficial y actual del Lenguaje de Marcado de Hipertexto (HyperText Markup Language), el estándar fundamental para crear y estructurar páginas web. Fue desarrollado por el consorcio internacional W3C y el grupo WHATWG, estableciéndose como norma oficial en 2014, con el objetivo de sustituir versiones anteriores y superar sus limitaciones. A diferencia de los lenguajes de programación, HTML5 no procesa lógica ni cálculos, sino que describe y organiza el contenido mediante etiquetas. Su gran innovación es el uso de etiquetas semánticas que indican claramente qué función cumple cada parte de la página, además de integrar capacidades que antes requerían complementos externos. HTML5 actúa como el esqueleto de cualquier sitio web, cumpliendo funciones esenciales para que el contenido se muestre, funcione y se adapte correctamente. Principales usos: - Estructurar el contenido: Organiza textos, imágenes, enlaces, listas y tablas en secciones lógicas, facilitando tanto la lectura para el usuario como el trabajo para los desarrolladores. - Integrar multimedia nativamente: Permite insertar audio, video y gráficos vectoriales directamente en la página, sin necesidad de instalar complementos como Flash. - Crear elementos interactivos: Facilita el diseño de formularios avanzados, botones, controles de reproducción y animaciones que responden a las acciones del usuario. - Habilitar funcionalidades modernas: A través de sus APIs integradas, permite almacenar datos en el dispositivo, trabajar sin conexión a internet, detectar la ubicación geográfica, arrastrar y soltar elementos, entre otras opciones. - Garantizar la adaptabilidad: Es la base técnica para que las páginas se ajusten automáticamente a celulares, tabletas, computadoras o pantallas grandes, manteniendo una experiencia de calidad. 3. ¿Por qué es importante en la creación de páginas web? HTML5 marcó un punto de inflexión total en el desarrollo web, convirtiéndose en el pilar indispensable sin el cual no existiría la internet moderna que usamos hoy.

Los formularios son elementos fundamentales de interacción entre el usuario y el sitio web. Se trata de conjuntos de campos y controles que permiten recopilar información, ya sea para enviarla a un servidor, procesarla en el mismo navegador o realizar acciones específicas como buscar, registrarse, comprar o contactar. Son la forma más común de intercambiar datos en internet: desde un registro de usuario hasta un formulario de pago o una encuesta. En HTML5, los formularios han evolucionado mucho, incorporando nuevos tipos de campos y validaciones automáticas que facilitan tanto el trabajo de los desarrolladores como la experiencia de los usuarios, sin necesidad de códigos externos complejos. Sus atributos más importantes son: -  action : Indica la dirección o archivo al que se enviarán los datos recopilados. -  method : Define el método de envío, generalmente  GET  (para consultas simples) o  POST  (para información sensible o voluminosa).Aquí el alumno debe investigar y explicar qué son los formularios y cuáles son algunas etiquetas y elementos como form, label, input, textarea, select y button. 2.  <label>  Es la etiqueta que etiqueta o describe cada campo del formulario, indicando al usuario qué información debe ingresar. Atributos principales: -  controls : Muestra los botones de reproducción visibles para el usuario. -  autoplay : Inicia el sonido automáticamente (se usa con moderación). -  loop : Repite el audio en bucle continuo 3.
